# pokemon-french 
I got inspired by [sindresorhus](https://github.com/sindresorhus/pokemon)
    
## Install

```
$ npm install --save pokemon-french
```

## Usage

```js
const pokemon = require('pokemon-french');
console.log(pokemon[0]); // Bulbizarre
```
